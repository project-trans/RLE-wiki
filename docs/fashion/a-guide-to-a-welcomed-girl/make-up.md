# 0x06 化妆的基础是护肤
> 嘿，大家好，我是雨曦，我来给大家讲一讲关于护肤的事。

## 01 护肤的意义&明白自己究竟要做什么

在讲护肤的内容之前，我想问问大家，我们为什么要护肤呢？作为跨性别女生，或许各位大都对自己的容貌或者能够“pass”有一定的追求。容貌的一部分来自于五官、轮廓，另一部分就来自于皮肤了。我们想想看，顺女的皮肤是不是一般都比男生的皮肤更细腻、白皙、有光泽、有弹性呢？让自己的皮肤变得更好是让自己更好看，更接近顺女外观的一步。
顺性别女生的皮肤好，除了性激素的一部分影响外，她们更注重护肤是非常重要的一点，护肤不一定让皮肤变得特别好，但良好、正确的护肤可以让皮肤处于一个比较好的状态里。

- 什么是比较好的状态呢？

  - 嗯，依我的经验，皮肤有较少的问题（如痘痘、泛红、起皮、干燥、刺痛等）；并且皮肤光泽度比较好，也可以说是不暗沉，这不代表肤色白，肤色是天生决定的；皮肤紧致有弹性。如果能做到这些，说明皮肤已经处于一个比较好的状态了。
  为什么护肤可以让皮肤变好？（不感兴趣的小伙伴可以跳过这一部分，这部分只是说明护肤的原理。）

  - 有些人可能会发现，为什么我脸上会长痘，会起皮、干痒、有皱纹，觉得自己脸上的皮肤是全身最差的皮肤。导致这样的结果有三大主要原因一、脸常年暴露在外，受到紫外线的照射较多，导致出现皱纹、斑点、暗沉这些问题。二、脸上的毛孔和皮脂腺很多，容易产生痘痘、黑头、毛孔粗大这些问题。三、脸部皮肤很薄、很脆弱，容易产生泛红、刺痛、干燥等问题。护肤中最重要的保湿和防晒就是为了减少这三个特点对皮肤状态的负面影响。防晒可以大大减少紫外线对皮肤的伤害，而保湿可以让干性皮肤更好的保持住皮肤中的水分，让皮肤保持在有光泽、饱满、不容易产生皱纹的状态，保湿又可以让油性皮肤减少出油，减少痘痘、黑头、毛孔堵塞出现的概率，同时也起到锁水的作用。
  
- 除了让皮肤减少出现问题的概率外，护肤还有什么作用呢？

  - 首先是抗老，让皮肤看起来处于一个比较年轻的状态，比如30岁看起来像22，3岁，40岁看起来像30岁。抗老是包含很多方面的，比如减少皱纹、皮肤紧致、毛孔变小、皮肤更细腻有光泽。衰老是不可避免的，但我们可以让衰老来的慢一些，让自己的青春留的久一些。我个人因为开始hrt和rle的时间比较晚（大概率比你们现在的年纪还要大），所以对于女孩子的青春，皮肤状态的保持执念比较深，对于抗老方面我是比较看重的。
  
  - 第二是美白，可以让皮肤看起来白皙一些，使用美白产品不会改变原有的肤色，但能让被晒黑、熬夜暗沉的肤色恢复原有的状态。

- 什么时候开始护肤呢，每个年龄段需要做好的护肤需求是什么呢？

  - 我的看法是6岁就可以进行防晒了，12岁就可以开始护肤了。

  - 6-11岁做好防晒，12-20岁做好清洁、保湿、防晒，21-30岁做好清洁、保湿、防晒、抗氧化、抗初老，31岁+做好清洁、保湿、防晒、抗氧化、抗老。
  
- 那如果年纪大了还没开始护肤，会不会已经晚了，没用了呢？

  - 护肤从什么时候开始都不晚，是一个日积月累的过程，我从25岁开始才有好好护肤，皮肤状态也保持的不错，皮肤年龄看起来和20岁差不多
