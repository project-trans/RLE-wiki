## 护肤

### 维生素C、E抗氧化

我们可以通过口服维生素C和口服维生素E进行一些抗氧化

考虑到大部分跨性别者经济能力有限，我个人建议直接买东北制药的维生素C片，其他的什么维生素C泡腾片，各种自然维生素C和这个效果都一样。但价格贵的多，还是这个维生素C片，价格实惠点。

维生素E，又名生育酚，也具有很强的抗氧化性，维生素E经常被添加到各种护肤品中，比如像有协和维E乳，口服维生素E效果也不差。

维生素C价格一般五元左右，维生素E价格十元左右，这两个可以刷医保卡，如果有医保卡的话，可以有一定优惠，有的地方可以做统筹，这样对经济友好点。

### 烟酰胺/维生素B3

维生素B3具有一定，延缓衰老，保水补湿，美白祛斑的作用。可以买一瓶复合维生素B片，里面含有一定的维生素B3

药店有卖，价格一般五元到十元不等，如果有医保的话，可以刷医保卡，这样子便宜一点，可以减轻一下自身经济压力。

::: tip 温馨提示
以上药品均属非处方药，但是还是建议使用前咨询专业医师或药师，这里仅表达我个人建议，不代表具体医疗意见，具体医疗意见还请咨询专业医师或药师。
:::

### 补水

#### 补水喷雾

市面上有很多补水喷雾，但个人不太推荐，比如常见的依云补水喷雾，其实有个平替的办法，买一瓶依云矿泉水（最好去免税店买，让便宜点）装在喷瓶里面往脸上喷

这个矿泉水300多毫升，一般超市有卖，五元一瓶左右

这种矿泉水里面装的水和补水喷雾，里面装的水成分一样，都是依云矿泉水，自己买个喷瓶，装进去就可以了，效果是一样的，可能因为价格原因导致心理差异，产生了一些细微的感觉，但这些可以忽略不计。依云矿泉水补水喷雾之所以好，是因为里面含有一定的锶元素，锶元素可以使我们的皮肤更有弹性

如果要比依云矿泉水价格更低，还可以使用屈臣氏蒸馏水，或者娃哈哈矿泉水

屈臣氏矿泉水500毫升，在屈臣氏大概三元左右

当然也可以使用娃哈哈纯净水，娃哈哈纯净水和屈臣氏蒸馏水，这两个都是比较纯的，在实验室都是可以使用的。

娃哈哈矿泉水很多，小卖部都有，价格便宜，一块五一瓶左右

很多人使用补水喷雾都有一个误区，就是直接往脸上喷，我们需要注意的是，喷的时候要距离脸部15到20厘米，并且拍完后用手拍打按压帮助吸收，如果不拍打的话，会使我们脸部因水蒸发而失去补水效果，甚至是让脸变得更干。

推荐使用的，不只有依云矿泉水、屈臣氏蒸馏水、娃哈哈纯净水。可以使用下丝瓜薏仁水，一般15元左右，500毫升

#### 芦荟胶

芦荟胶选择主要看成分表，尽量使库拉索芦荟排名靠前，我个人比较爱用安安金纯的芦荟胶

在这里提一下使用方法，芦荟胶虽然有很强的补水保湿效果，使用时一般建议涂一层在脸上，如果有痘痘的地方，着重厚涂一下，一般涂了之后，五个小时之后，建议洗下脸，因为芦荟胶一般不容易被吸收，如果敷在脸上，时间过久，会有大量的细菌繁殖，这样会增加我们长痘的几率。

如果有伤口，结痂之后，也可以涂一下芦荟胶，这样子保持伤口湿润，有利于恢复，也不容易留下印记（以上所述，仅针对伤口比较浅的情况，情况严重，请咨询医生）

## 减肥

先说我们对于健康体重的定义，

Bmi值在大于等于18，小于等于24之间，属于健康体重。Bmi值计算方式为，体重除以身高的平方。体重单位取千克，身高单位取米。

::: tip 温馨提示
希望大家不要过度减肥，合理膳食，不偏食，不挑食。
:::

一般我建议在吃饭前先喝一杯水，吃饭的时候先吃菜，同时多吃点蔬菜，最后吃米饭。这样子，饱腹感会很强，但不会摄入太多的能量。

同时运动的时候建议以有氧运动为主，第一是有氧运动可以消耗大量的脂肪，第二是有氧运动对身体机能要求没那么高，同时，有氧运动不会产生大量的乳酸堆积，这样子不会让我们的腿很痛。

建议每次吃饭时用筷子尖一厘米接触食物，这样子，你每次吃的东西很少，吃的也很慢，这样子就是为了减小你的死食物摄入量，起到一个减肥的效果，同时，这样子行为举止也比较优雅，也比较接近女孩子的气质。

## 脱毛

### 机械脱毛

主要使用修眉刀或剃须刀，个人建议在进行脱毛时，把皮肤表面用水打湿，最好再涂一层肥皂液，这样子起到润滑的作用，防止割伤。

建议直接用肥皂在需要脱毛部位进行厚涂一层，厚涂好的标准是，欲刮毛部位的毛发，已经贴附在体表。进行润滑之后，不容易割出血。

同时，个人建议使用修眉刀，而不是剃须刀，剃须刀比较锋利，一旦割伤，容易一次性割掉一大块皮，而修眉刀如果使用不慎，可能只是割出一道长长的划痕。修眉刀的危害要小一点。

同时建议用修眉刀去除体毛的时候，刀尖翘起来一点，修眉刀割伤一般是刀尖哥伤，所以说这点要注意。

### 激光脱毛

我在家自行使用激光脱毛仪，建议脱毛前先涂一点防晒霜，因为脱毛仪会发出大量高能射线，容易导致低温烫伤，同时，也有紫外线，这也会加速我们皮肤老化。

如果是去医院进行激光脱毛，具体情况请咨询专业医师，但还是建议在激光脱毛前涂一些防晒霜。

### 脱毛膏

先在一些不明显位置进行试验，防止过敏现象产生，不建议在脸部这些位置使用脱毛膏，脱毛膏对皮肤也有一定的刺激，不建议长时间使用。

## 洁面

洁面乳在此不具体推荐，根据每个人不同肤质进行选择，但是在这里跟大家讲一下洁面的误区。

洁面乳，洗面奶等不要直接往脸上涂，建议先用起泡网打出绵密泡沫后再涂到脸上。因为在起泡过程中需要消耗一定的水分，如果在脸上起泡，就消耗的是我们脸上的水分，不仅起不到洁面的作用，还会使我们的脸变得非常的干，所以这个时候我们就需要起泡网的辅助。

## 毛孔问题

针对毛孔粗大问题，如下所示进行解决

1. 用毛巾清水洁面
2. 用纯净水（推荐屈臣氏蒸馏水，娃哈哈饮用纯净水）对毛孔粗大部位进行湿敷，时长不要超过三分钟。时间过长，可能会破坏水油平衡。
3. 用小型空气加湿器对毛孔粗大部位进行冷喷，冷喷大约15至20分钟。 \
  以上两步都可以，但是需要坚持，至少坚持一个月以上
4. 洁面后如果是早上就涂一点壬二酸，在毛孔粗大部位，用手进行按摩，5到10分钟之后清洗掉。壬二酸有一定的平衡，油脂溢出功能。 \
  如果是晚上，在洁面后涂一点水杨酸在毛孔粗大部位，用指腹进行按摩，5至10分钟之后清洗掉。杨酸有软化油脂，疏通毛孔，抑制油脂的功能。
5. 在涂完壬二酸或水杨酸后，湿敷一层芦荟胶，进行保湿处理。芦荟胶选择见前文所述，但芦荟胶最好不要长时间敷在皮肤上，同时用手指拍打吸收，一段时间后洗脸擦去芦荟胶。

::: tip 温馨提示
水杨酸和壬二酸，建议一周使用2到3次，在药店有卖，具体使用方法，请咨询专业医师或药师，如有不适，请立即停止使用，并前往专业皮肤科医生处就诊。同时有医保的可以刷一下医保卡，这样子减轻一下大家的经济压力。

个别人可能有壬二酸，水杨酸不耐受的情况，刚开始使用时可能有刺痛的感觉，一周以后逐渐好转，如一周后没有好转，请停止使用并咨询专业医师。
:::

:::tip 温馨提示
在化妆品/护肤品使用过程中，如有红肿过敏等不适，请立即停止使用，并咨询专业皮肤科医生。
:::

## 口腔问题

### 牙齿

#### 牙齿美白

如果牙齿比较黄，正常的牙齿是会稍微泛点黄，因为牙釉质是透明的，透过牙釉质看到里面的一些齿质。如果黄的特别厉害，可以改用牙粉代替牙膏进行刷牙。

::: tip 温馨提示
在牙粉使用过程中，如有口腔不适，请立即停止使用，并咨询专业医师
:::

#### 正畸

如果牙齿不太整齐，可以选择正畸进行调整，因为人面部70%的延迟是由牙齿决定，并且牙齿健康也很重要。具体正畸方案请咨询专业正畸科医生，在这里提醒下大家，在征集过程中注意的一些情况。

1. 正畸过程中经常会所谓出现的牙套脸，是因为咬肌长期不咬，硬东西萎缩导致的，这个过程中可以买一些咬胶来进行咀嚼，或者嚼口香糖也可以。
2. 正畸过程中，尽量吃软一点，稀一点的东西，避免把托槽磕掉 。过程也可以当做减肥，尽量避免吃甘蔗，排骨这一类硬的东西，多吃粥，这一类的。
3. 尽量选择专业的正畸科医生，正畸是硕士学科，一般的口腔小诊所是没有正畸资格的，
4. 定期复诊

有的正畸在有的地区是可以刷医保卡，建议早一点做，在学生时期做。最好有家长的支持。

::: tip 温馨提示
以上仅表个人建议，具体正畸方案及正畸实施流程及医疗过程，请咨询专业正畸科医生，以上仅表个人建议
:::
